IMAGE TO PDF CONVERTER

HOW TO USE:

1 Double-click "Run" or the shortcut 
2 The program window will open
3 Click "Select Images" to choose your JPG/JPEG files or just drag and drop them into
4 Click "Create PDF" to convert them to a PDF file, right bottom

REQUIREMENTS:

- Java must be installed on the computer
- Download Java from: https://www.oracle.com/java/technologies/downloads/

FILES INCLUDED:

- Image to PDF Converter.lnk  -> Shortcut (double-click this to run)
- Run Image to PDF Converter.bat -> The launcher script
- target/jpg-to-pdf-gui.jar    -> The actual program
- test1.jpg, test2.jpg, test3.jpg -> Sample images to test with
